# 이미지 폴더

이 폴더에는 오달동이 제품 이미지가 저장됩니다.

## 필요한 이미지 파일

다음 이미지 파일들을 이 폴더에 추가해주세요:

1. **soap.jpg** - 해파랑 비누 제품 이미지
   - 권장 크기: 800x600px 이상
   - 형식: JPG, PNG

2. **spray.jpg** - 해파랑 스프레이 제품 이미지
   - 권장 크기: 800x600px 이상
   - 형식: JPG, PNG

3. **salt.jpg** - 해파랑 소금 제품 이미지
   - 권장 크기: 800x600px 이상
   - 형식: JPG, PNG

## 이미지 가이드라인

- **파일 크기**: 500KB 이하 권장 (웹 최적화)
- **해상도**: 고해상도 이미지 사용
- **비율**: 4:3 또는 16:9 권장
- **배경**: 깔끔한 단색 배경 또는 자연스러운 배경

## 이미지 최적화 팁

이미지를 추가하기 전에 다음 도구들로 최적화하세요:

- [TinyPNG](https://tinypng.com/) - PNG/JPG 압축
- [Squoosh](https://squoosh.app/) - 구글 이미지 최적화 도구
- [ImageOptim](https://imageoptim.com/) - Mac용 이미지 최적화

현재는 이미지가 없어도 웹사이트가 정상적으로 작동하며, 
이미지 로드 실패 시 플레이스홀더가 표시됩니다.
